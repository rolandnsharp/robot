Robot Test
==========

# Getting Started

```
npm install -g robot-test
```
# Usage

Simply type `robot` to start the program.

From here you will be given a command prompt for your robot.

# Valid Commands

- PLACE X,Y,F (0-4, 0-4, NORTH/EAST/SOUTH/WEST)
- MOVE
- LEFT
- RIGHT
- REPORT

# Testing

```
npm test
```
